# PlayerMax Labs

A simple marketing site for PlayerMax Labs built with Next.js.

## What this project includes

- Homepage with athlete/coach view switch
- About page
- Waitlist page with a form
- Shared navbar across pages

## Tech stack

- Next.js (App Router)
- React
- Tailwind CSS v4
- Framer Motion
- Lucide icons

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. Open:

```text
http://localhost:3000
```

## Scripts

- `npm run dev` - start local dev server
- `npm run lint` - run ESLint
- `npm run build` - create production build
- `npm run start` - run production server

## Main routes

- `/` - landing page
- `/about` - about page
- `/waitlist` - waitlist form page

## Project structure

```text
src/app
src/components
```

`src/app` contains pages/routes.  
`src/components` contains reusable UI sections.
